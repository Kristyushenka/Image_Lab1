ЛАБОРАТОРНАЯ РАБОТА 1
Работа с изображениями
=================================================================

Написано по критериям на 50 баллов

ФУНКЦИОНАЛЬНОСТЬ:
- Добавление неограниченного количества изображений
- Режимы наложения: Normal, Sum, Difference, Multiply, Screen, 
  Average, Min, Max, Overlay, SoftLight, HardLight
- Регулировка прозрачности каждого слоя (0-100%)
- Перемещение слоёв вверх/вниз (порядок наложения)
- Удаление слоёв
- Предпросмотр результата в реальном времени
- Сохранение результата в форматах PNG, JPEG, BMP
- Поддержка изображений разного размера

КАК ПОЛЬЗОВАТЬСЯ:
1. Запустите программу
2. Нажмите "Добавить изображение" и выберите один или несколько файлов
3. Изображения появятся в списке слоёв слева
4. Выберите слой в списке для редактирования его настроек
5. Выберите режим наложения из выпадающего списка
6. Отрегулируйте прозрачность ползунком
7. Используйте кнопки "↑ Вверх" и "↓ Вниз" для изменения порядка слоёв
8. Нажмите "Обновить" для перерисовки результата
9. Нажмите "Сохранить результат" для сохранения итогового изображения

ТРЕБОВАНИЯ:
- .NET Framework 4.8 и выше
- Библиотека System.Drawing.Common

СТРУКТУРА ПРОЕКТА:
- ImageLayer.cs - класс слоя изображения
- Form1.cs - основная форма приложения
- Form1.Designer.cs - дизайн формы
- Program.cs - точка входа приложения
